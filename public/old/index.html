  • ZFS for Proxmox Storage

    Using ZFS with Proxmox Storage has never been easier! Resist the urge to rename your zfs datastores that are being used for VMs. Snapshots on VMs (these would sync in a cluster) should be stored on a 'local-zfs' datastore. Using the same name across devices allows Proxmox to sync these stores if there was ever to be a cluster formed. And it allows for uniform naming.
